# fastapi-template

Production-ready FastAPI template with clean service boundaries, JWT auth, and tested API flows.

## Stack

- FastAPI + Pydantic
- PyJWT + python-dotenv
- Ruff + mypy
- Pytest + coverage
- uv (package manager)

## Endpoints

- `GET /health`
- `GET /docs`
- `GET /v1/public`
- `GET /v1/customer`
- `GET /v1/auth/login`
- `GET /v1/private`

## JWT Contract

Login returns:

- Body: `{"token":"..."}`
- Header: `Authorization: Bearer <token>`
- Header: `X-JWT-Token: <token>`

## Commands

- `make install` — install production dependencies
- `make install-dev` — install dev dependencies
- `make run` — run the application
- `make test` — run tests with coverage
- `make docker-build` — build Docker image
- `make docker-test` — run tests in Docker
- `make docker-curl-test` — run curl integration tests in Docker

## Environment Variables

- `PORT=8000`
- `JWT_SECRET=your-super-secret-jwt-key-at-least-32-characters-long-for-hs256`
- `JWT_ALGORITHM=HS256`
- `JWT_EXPIRY_MINUTES=60`

## Architecture

```text
HTTP -> API Routes -> JWT Middleware -> Services -> Repositories -> Domain
```
